# plataforma-incidencias-ti
Evidencia 1

Este proyecto corresponde al desarrollo de una plataforma web básica para la gestión de incidencias de soporte técnico.
El sistema permite registrar, visualizar y filtrar incidencias de manera sencilla, simulando el funcionamiento de una herramienta real de mesa de ayuda.
Funcionalidades principales:
° Inicio del sistema mediante una página principal.
° Pantalla de inicio de sesión para técnicos.
° Registro de nuevas incidencias mediante un formulario.
° Listado de incidencias en una tabla dinámica.
° Filtros por estado (Abierta, En proceso, Cerrada).
° Filtros por prioridad (Alta, Media, Baja).
° Visualización de los datos principales de cada incidencia.
Tecnologias Utilizadas:
°HTML5 para la estructura del sestema
°Css3 para un diseño base
°Javascript para la logica 
°Github para subir el repositorio y control de versiones.
